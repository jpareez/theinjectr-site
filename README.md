# The InjectR — website

Static marketing site for The InjectR (Randa Tighe, RPN) — nurse-led medical aesthetics, Etobicoke and the GTA.

- **Stack:** static HTML / CSS / vanilla JS (designed in Claude Design, exported). No build step, no framework.
- **Deploy:** Cloudflare Pages — Framework preset: None, Build command: (blank), Build output directory: `/`, Production branch: `main`.
- **Domain:** theinjectr.com
- **Pages:** index, about, treatments, results, contact, privacy, terms.

Edits are committed directly; pushing to `main` triggers the Cloudflare Pages deploy.
